# Mental-Health-Mood-Tracker
A modular system designed to help individuals track their daily moods, detect patterns, identify anomalies, and receive actionable mental health insights. This tool empowers self-awareness and early support through simple data-driven analysis.

Features
Daily Mood Logging: Records mood levels on a scale of 1 (very low) to 10 (very high).

Trend Analysis: Calculates 7-day rolling averages to identify mood trends.

Anomaly Detection: Uses Isolation Forest to detect sudden mood changes or irregular patterns.

Insight Generation: Provides personalized feedback based on mood data (e.g., low mood streaks, downward trends).

Visualization: Generates intuitive plots showing mood over time with trend lines and anomalies.

Technologies Used
Python

Pandas, NumPy – Data processing and manipulation

scikit-learn – Machine learning model (Isolation Forest)

Matplotlib, Seaborn – Data visualization

How It Works
Mood entries are generated or collected.

Data is processed for trend analysis and anomaly detection.

Key insights are printed to the console.

Visual graphs highlight mood fluctuations and anomalies.

Use Cases
Personal mood tracking and journaling

Early detection of emotional patterns or burnout

Supplementary tool for therapists or mental health coaches
